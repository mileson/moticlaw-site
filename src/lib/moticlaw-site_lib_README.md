# moticlaw-site_lib 文件夹说明书

## 核心功能

提供官网共享的轻量工具函数、语言判断和发布清单转换逻辑。

## 输入

接收请求头语言值、浏览器语言偏好列表、OSS 发布 JSON 和页面需要的平台映射规则。

## 输出

输出供路由层和组件层共享的语言工具，以及统一发布数据模型。

## 定位

位于 `src/lib`，作为官网业务逻辑的基础工具层。

## 依赖

依赖 TypeScript、运行时 `fetch` 和官网当前支持的平台规则。

## 维护规则

- 新增工具模块、调整平台映射或更换数据源格式后，必须同步更新本说明书。
- 若工具层开始承载更复杂的业务逻辑，应同步更新 `docs/basic/backend-structure.md`。
