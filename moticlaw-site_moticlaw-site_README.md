# moticlaw-site 项目文件夹说明书

## 核心功能

承载 MotiClaw 官网站点的页面、发布下载入口、静态资源和部署配置。

## 输入

接收桌面项目发布到阿里云 OSS 的最新清单、浏览器语言偏好和本地开发构建命令。

## 输出

输出部署到 Vercel 的官网页面、`/api/releases/latest` 发布接口，以及公开安装脚本和静态资源。

## 定位

位于独立仓库根目录，是 MotiClaw 对外官网和下载页的唯一前端工程。

## 依赖

依赖 Next.js App Router、React 19、Tailwind CSS v4、阿里云 OSS 公网发布源和 Vercel 部署链路。

## 维护规则

- 新增核心目录、调整部署链路或变更官网职责后，必须同步更新本说明书。
- 发布数据源、下载入口或项目结构变化时，必须同步更新 `docs/basic/`。
